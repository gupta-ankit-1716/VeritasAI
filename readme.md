# VeritasAI | Advanced Deepfake Detection & Media Forensics

**VeritasAI** is a high-fidelity web application designed to restore digital integrity in an era of synthetic media. Built with a focus on **Cyber-Forensic aesthetics** and **Secure Session Management**, this platform provides a professional environment for analyzing media for neural inconsistencies.

---

## 🚀 Core Features

### 1. Multi-Layered Forensic Dashboard
* **Neural Scanning Animation:** A real-time progress tracking system with a live terminal log simulating frame-by-frame analysis.
* **Forensic History:** A persistent session-based table tracking past reports, verdicts (Authentic vs. Deepfake), and confidence scores.
* **Session Metrics:** Live uptime tracking and mock API credit management to simulate an institutional SaaS environment.

### 2. Secure Access Gatekeeper
* **Authentication Guard:** A custom JavaScript "Gatekeeper" that intercepts unauthorized access attempts to the Dashboard.
* **Session Persistence:** Utilizing `localStorage` to manage active forensic sessions without a traditional backend.
* **Secure Termination:** A one-click "Terminate Session" protocol that wipes local data and restores the gatekeeper.

### 3. Institutional Marketing Suite
* **Technical Breakdown:** A "How it Works" page explaining DCT (Discrete Cosine Transform) and Temporal Coherence.
* **Ethics & Manifesto:** A dedicated compliance page detailing forensic neutrality and privacy protocols.
* **Tiered Pricing:** A scalable API subscription model (Sentinel, Inquisitor, Oracle).

---

## 🛠️ Technical Stack

* **Frontend:** Semantic HTML5, CSS3 (Custom Variables, Flexbox, Grid, Glassmorphism).
* **Logic:** Vanilla JavaScript (ES6+), Event Delegation, Session Storage API.
* **Icons:** [Lucide-Icons](https://lucide.dev/) for high-end vector consistency.
* **Design:** Custom "Cyber-Dark" theme with #00E8FF (Veritas Cyan) accents.
